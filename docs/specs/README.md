# Specs

This directory holds durable product and version specifications for `aws-sftp-testbed`.

- `sftp-testbed-version-spec.md` describes the current V1 toolset, locked MVP decisions, non-goals, and possible V2/V3 expansion paths.

Specs are more durable than proposals. A proposal may describe how to build a feature; a spec records what the project is, what has shipped, and which future ideas are intentionally not current scope.
